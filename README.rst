Yubiauth user backend
---------------------
Adds Yubico Yubikey authentication support to Owncloud

* Requirements:
    * ``OwnCloud 4.5+``
    * ``php-yubico`` (included)

* Status:
    * ``"Alpha"`` -- Works well for me, but not really tested in other installations

* Changelog:
    * ``v0.0.2``
        * first version
        * tested with OC 4.5.x
